
# ML-Based Fertilizer Recommendation: Predicting Crop Response and Optimizing Fertilizer Type & Dosage

📖 **Read the full article:** https://www.ijteonline.in/assets/uploads/issue_pdf/1771004264.pdf (Search Article name in PDF)

## 📌 Overview
This repository contains a machine learning pipeline that bridges the gap between agronomy and economics. Instead of simply predicting crop yield, this project identifies the most profitable fertilizer application strategy for individual farming plots. By simulating different fertilizer types and dosages, the model calculates the exact point where the cost of additional fertilizer outweighs the financial benefit of the increased yield.

## 🚀 Key Features & Methodology
1. **Feature Engineering**: 
   * Calculates specific nutrient deficits (Nitrogen, Phosphorus, Potassium) based on baseline soil requirements.
   * Computes the fertilizer-to-crop price ratio to factor in fluctuating market costs.
2. **Predictive Modeling (LightGBM)**: 
   * Trains a `LightGBM` regression model on preprocessed soil metrics, crop types, and historical yield data to accurately predict `yield_kg_ha`.
3. **Economic Optimization Loop**: 
   * Simulates applying common fertilizer products (Urea, DAP, NPK 20:20:0) across a grid of dosages (0 to 300 kg/ha).
   * Evaluates the projected profit for every scenario (Yield × Crop Price - Fertilizer Cost).
   * Recommends the specific product and dose combination that maximizes net financial return for the farmer.

## 🛠️ Dependencies
Ensure you have the following Python libraries installed before running the script:
```bash
pip install pandas numpy scikit-learn lightgbm matplotlib

```

## 💻 Usage

To run the optimization pipeline, ensure `fert_opt_synthetic.csv` is in the same directory as the script, then execute:

```bash
python fertilizer_optimization.py

```

## 📂 Output Files

The script creates a new folder called `outputs_fert_opt` containing:

* **`recommendations.csv`**: The final dataset containing the best recommendation for each plot (Plot ID, Product Name, Dose, Predicted Yield, and Maximum Profit).
* **Saved PNG Plots**: The visual distributions of the model's performance and economic outputs.
